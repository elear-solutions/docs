Zero-Lag Media Edge
===================

In this section, we introduce you to our Zero-lag Media Edge platform tailored to use cases like Watch Together, which includes support for Media Calls with multiple participants & support for exchanging video and audio; the ability to share OTT content in Media Calls; and upcoming support for other media types like text and files.
